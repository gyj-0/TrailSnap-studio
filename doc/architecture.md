## 2. 服务详解

### 2.1 前端服务 (package/website)

**技术选型**: Vue 3 + TypeScript + Vite

**主要功能**:
- 用户界面展示
- 照片上传与浏览
- 相册管理
- 地图轨迹展示
- 票据识别结果展示

**目录结构**:
```
src/
├── api/           # 后端接口封装 (Axios)
├── components/    # 通用组件
├── composables/   # 组合式函数 (Hooks)
├── layouts/       # 页面布局
├── router/        # 路由配置
├── stores/        # Pinia 状态管理
├── types/         # TypeScript 类型
└── views/         # 页面视图
```

### 2.2 后端服务 (package/server)

**技术选型**: FastAPI + SQLAlchemy + PostgreSQL

**主要模块**:

| 模块 | 职责 | 说明 |
|------|------|------|
| `api/` | API 路由 | RESTful API 端点定义 |
| `core/` | 核心配置 | 日志、配置、安全 |
| `crud/` | 数据访问 | 数据库 CRUD 操作 |
| `db/` | 数据库 | 模型定义、会话管理 |
| `schemas/` | 数据模型 | Pydantic 请求/响应模型 |
| `service/` | 业务服务 | 复杂业务逻辑实现 |
| `utils/` | 工具函数 | 通用工具类 |

**关键特性**:
- 异步处理（async/await）
- 依赖注入
- 自动 API 文档 (OpenAPI/Swagger)
- JWT 认证
- 数据库迁移 (Alembic)

### 2.3 AI 微服务 (package/ai)

**技术选型**: FastAPI + PaddleOCR + PyTorch + InsightFace

**主要功能**:

| 功能 | 技术 | 说明 |
|------|------|------|
| OCR 识别 | PaddleOCR 3.3.2 | 多语言文本检测与识别 |
| 人脸识别 | InsightFace | 人脸检测、特征提取、比对 |
| 目标检测 | YOLO + PyTorch | 通用物体检测 |
| 票据识别 | 自定义模型 | 火车票、发票等票据结构化 |

**模型管理**:
- 懒加载机制：按需加载模型
- LRU 缓存：最近使用的模型保持在内存
- 资源释放：空闲模型自动卸载

## 5. 接口设计

### 5.1 RESTful API 规范

- 基础路径: `/api/v1/`
- 响应格式:
  ```json
  {
    "code": 200,
    "message": "success",
    "data": { ... }
  }
  ```

### 5.2 主要端点

| 模块 | 端点 | 说明 |
|------|------|------|
| 认证 | `POST /api/v1/auth/login` | 用户登录 |
| 认证 | `POST /api/v1/auth/register` | 用户注册 |
| 用户 | `GET /api/v1/users/me` | 当前用户信息 |
| 相册 | `GET /api/v1/albums` | 相册列表 |
| 相册 | `POST /api/v1/albums` | 创建相册 |
| 照片 | `GET /api/v1/photos` | 照片列表 |
| 照片 | `POST /api/v1/photos/upload` | 上传照片 |
| 照片 | `GET /api/v1/photos/{id}` | 照片详情 |

### 5.3 AI 服务接口

| 功能 | 端点 | 说明 |
|------|------|------|
| OCR | `POST /ocr/recognize` | 通用文字识别 |
| OCR | `POST /ocr/detect` | 文本检测 |
| 人脸 | `POST /face/detect` | 人脸检测 |
| 人脸 | `POST /face/compare` | 人脸比对 |
| 票据 | `POST /tickets/recognize` | 票据识别 |

## 6. 安全设计

### 6.1 认证授权
- JWT Token 认证
- 访问令牌（15分钟有效期）
- 刷新令牌（7天有效期）
- 密码加密（bcrypt）

### 6.2 数据安全
- HTTPS 传输
- SQL 注入防护（SQLAlchemy ORM）
- XSS 防护
- CSRF 防护

### 6.3 文件安全
- 文件类型白名单
- 文件大小限制
- 病毒扫描（ClamAV 集成点）

## 7. 性能优化

### 7.1 前端优化
- 路由懒加载
- 组件异步加载
- 图片懒加载
- 虚拟滚动（长列表）

### 7.2 后端优化
- 数据库连接池
- Redis 缓存
- 异步 IO
- 分页查询

### 7.3 AI 服务优化
- 模型量化
